# SLTools

> Community Tools for StarLoco Retro

SLTools est un projet open source destiné à simplifier la création, la maintenance et l'évolution des serveurs StarLoco Retro.

Notre objectif n'est pas de remplacer StarLoco.

Notre objectif est de fournir les outils qui lui manquent.

---

## Philosophie

> **Tout ce que nous faisons doit pouvoir servir à la communauté.**

---

## Objectifs

- Faciliter la maintenance des serveurs
- Réduire les tâches répétitives
- Fournir des outils modernes
- Documenter les connaissances
- Encourager le partage

---

## Modules prévus

- House Scanner
- House Editor
- Scripted Cells Editor
- SQL Merge
- Map Viewer
- Zone Builder

---

## État du projet

🚧 En cours de conception.

Le développement débutera une fois le cahier des charges terminé.
