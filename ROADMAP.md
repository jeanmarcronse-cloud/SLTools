# SLTools - Roadmap

> *Une évolution progressive, réfléchie et documentée.*

---

# Vision

SLTools a pour objectif de fournir une suite d'outils modernes destinés à simplifier la création, la maintenance et l'évolution des serveurs StarLoco Retro.

Le projet suivra une règle simple :

> **Une fonctionnalité terminée vaut mieux que dix fonctionnalités commencées.**

Chaque version devra être stable, documentée et réellement utile.

---

# Foundation

Avant toute ligne de code.

* Mise en place du dépôt GitHub
* Documentation du projet
* Définition de l'architecture
* Cahier des charges
* Philosophie du projet

---

# Version 0.1 — House Scanner

Objectif :

Créer un outil capable d'analyser automatiquement les maisons.

Fonctions prévues :

* Détection des maisons cassées
* Vérification des MapID
* Vérification des CellID
* Rapport détaillé
* Export des anomalies

---

# Version 0.2 — House Editor

Objectif :

Permettre la correction des maisons sans modifier manuellement la base de données.

Fonctions prévues :

* Modification des MapID
* Modification des CellID
* Prévisualisation
* Sauvegarde automatique
* Historique des modifications

---

# Version 0.3 — Scripted Cells Editor

Objectif :

Créer un éditeur dédié aux `scripted_cells`.

Fonctions prévues :

* Lecture des transitions
* Modification graphique
* Vérification des incohérences
* Import / Export SQL

---

# Version 0.4 — SQL Merge

Objectif :

Fusionner intelligemment des scripts SQL provenant de différentes sources sans écraser les données existantes.

Fonctions prévues :

* Comparaison
* Fusion
* Détection des conflits
* Sauvegarde automatique

---

# Version 0.5 — Map Viewer

Objectif :

Visualiser facilement les cartes et leurs relations.

Fonctions prévues :

* Affichage des cartes
* Affichage des transitions
* Recherche rapide
* Navigation simplifiée

---

# Version 1.0 — Zone Builder

Objectif :

Fournir un environnement permettant de construire de nouvelles zones tout en automatisant les tâches répétitives.

Le but n'est pas de reproduire uniquement le Monde des Douze, mais de permettre à chacun de créer son propre univers.

---

# Les idées

Toutes les nouvelles idées seront étudiées avant leur développement.

Aucune fonctionnalité ne sera ajoutée sans validation du cahier des charges.

Les idées futures seront conservées dans le fichier `IDEAS.md`.

---

SLTools est développé avec une seule ambition :
